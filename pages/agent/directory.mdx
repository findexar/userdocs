# Directory

The **Directory** tab is the **global app catalog** on Agentsyx: search **across creators**, see apps **grouped by creator**, and **open** any listed app.

## Search

- Type in the **search** field and press **Enter** or click **Search**.
- Results (or the default **Known apps** list when the query is empty) load below.

## Results

Apps are grouped under **creator** names. Each app link sends you to that app’s path on agentsyx.com (similar to opening a **deep link**). You may arrive in the **[App](/agent/app-panel)** tab or the creator’s default entry point.

## Current app highlighting

When you’re already inside a creator’s app, the list may **highlight** the matching creator or app so you can see where you are in the catalog.

## Discovery vs Config

- **Directory** — Find **new** apps from **many** creators.
- **[Config](/agent/config)** — Turn apps you’re **entitled to** on or off **for this agent**.

Open something in Directory, then use **Config** if you want its **tools** available every time you chat.
